# 🎬 Netflix Data Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. The analysis uncovers trends in content, ratings, genres, countries, and release years to gain meaningful insights into Netflix's content library.

## 🎯 Objectives
- Clean and preprocess the Netflix dataset.
- Analyze the distribution of Movies vs TV Shows.
- Explore content ratings and genres.
- Identify top content-producing countries.
- Analyze yearly content additions.
- Generate business insights through visualizations.

## 🛠️ Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib

## 📂 Dataset
- netflix_titles.csv

## 📈 Key Insights
- Compared the number of Movies and TV Shows.
- Identified the most common content ratings.
- Analyzed top genres available on Netflix.
- Found the countries with the highest content production.
- Studied content release trends over the years.
